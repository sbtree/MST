MST: Metronix Servo Tester

Comm: components for communication, e.g. USB, RS232, CANOpen, IEEE488
Config: components for configuration, e.g. Ini-Reader
Database: interface to database
Device: components for measurement device, e.g. Multimeter, Oscilloscope, Thermometer
External: external components, e.g. libraries of the third part 
Function: functions, which are called in test scripts
General: fundamental procedures and functions for other modules  
Product: components for servo-products of Metronix
Runner: components for running a test
Script: components for reading and checking test scripts
